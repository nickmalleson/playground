# Schelling Embeddings

A model that uses text embeddings to define agents in a simple Schelling model

Published as: [Foundation Models and Embeddings for Nuanced Perception and Decision-Making in Agents](http://www.nickmalleson.co.uk///p/2025-08-SSC-Schelling_Embeddings.html). Presentation to the [Social Simulation Conference (SSC) 2025](https://ssc2025.tbm.tudelft.nl/), 25-29 August, Delft.
